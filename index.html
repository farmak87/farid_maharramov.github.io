PyTaSQL Part 1:
write credit simulator on Python

This project is a result of a discussion I had with a friend of mine about the scarcity of well-structured open data that we would want to store in MySQL tables and visualize using Tableau. So, we decided to generate pseudo-data ourselves by writing a credit simulator that will mimic a service that is usually being rendered by mobile network operators. Customers or subscribers of mobile network operators use mobile services in advance with further repayment. The purpose of the simulator is a generation of pseudo customers who make pseudo credit request and then pay them back by recharging pseudo-top-ups.
